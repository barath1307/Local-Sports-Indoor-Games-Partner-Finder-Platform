# Backend Authentication System

## Introduction
A backend authentication system built using Node.js, Express.js, MongoDB Atlas, JWT, and bcrypt.

## Technologies Used
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT
- bcryptjs

## Features
- User Registration
- User Login
- Password Hashing
- JWT Authentication
- Protected Profile Route

## API Endpoints

### Register
POST /auth/register

### Login
POST /auth/login

### Profile
GET /auth/profile

## Run Project

Install dependencies:

```bash
npm install
```

Start server:

```bash
node Server.js
```

Server:

```text
http://localhost:5000
```
